## Project: Blog App with Routing

Build a complete blog application with:

### Features
1. **Home page** - List of blog posts
2. **Post pages** - Individual post display
3. **Author pages** - Author profile and posts
4. **Search** - Search posts by title/content
5. **Categories** - Filter by category
6. **Comments** - Post comments (bonus)
7. **Admin panel** - Create/edit posts

### Pages
- `/` - Home with all posts
- `/post/:id` - Individual post
- `/author/:id` - Author page
- `/category/:name` - Category filter
- `/search` - Search results
- `/admin` - Admin panel (protected)

### Requirements
- React Router for navigation
- Dynamic routes with params
- Query parameters for search
- Protected admin routes
- API integration
- State management
- Error handling
- Loading states

### Tech Stack
- React + React Router
- State management (Context or Redux)
- Axios/Fetch for API
- CSS/Tailwind for styling

### Checkpoint
- [ ] Routing set up
- [ ] All pages created
- [ ] Navigation working
- [ ] Forms functional
- [ ] API integrated
- [ ] Error handling
- [ ] Responsive design
